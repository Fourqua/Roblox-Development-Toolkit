# Roblox Developer Toolkit

> This toolkit is meant to help passionate roblox developers to assist their systems.

---

## Structure

```
roblox-dev-toolkit/
│
├── README.md
│
├── SystemName/
│   ├── README.md
│   └── SystemName.lua
│
└── ...
```

---

## How to Use

1. Find the folder for the system you need.
2. Read its `README.md`.
3. Copy the code into your Roblox Studio project.
4. Follow any setup steps noted in the documentation.

---

## Contributing

- Create a new folder in `PascalCase`
- Add your `.lua` source file
- Add a `README.md` using the format below
- Open a pull request

### Module README Format

```
# System Name

One-line description.

---

## Overview
## Setup
## API Reference
## Examples
## Notes
```
